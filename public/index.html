YouTube Setup Guide
1. Go to YouTube Studio → Go Live → Stream
2. Copy your Stream Key and paste it in your slot
3. Set stream type to "Persistent" for 24/7 looping
4. Hit Start Streaming — StreamLoop will loop your videos automatically via RTMP/FFmpeg
⚠️ Real RTMP streaming requires a backend server (Node.js + FFmpeg). This tool generates the correct FFmpeg command for your setup. Copy it to run on your server.
